#################
Tutorials - Basic
#################

This chapter revolves around the :py:mod:`easygopigo3` module and just a bit around :py:mod:`di_sensors`.

Please make sure you have followed all the instructions found in :ref:`Getting Started <getting-started-chapter>` before jumping into tutorials.
In all these tutorials, you will need:

  1. A `GoPiGo3`_ robot.
  2. A sensor/actuator if requested in the tutorial : i.e.: a `Grove Buzzer`_, a `Line Follower`_, etc.
